| bucket | 用途 | RLS policy |

**建表 + 授权 SQL**（已经跑过, 文档留底）:
```sql
-- gallery 表
CREATE TABLE IF NOT EXISTS public.gallery (
  id bigint generated by default as identity primary key,
  city text, year text, url text not null,
  note text, created_at timestamptz default now()
);
alter table public.gallery enable row level security;
create policy "公开读" on public.gallery for select to anon using (true);
create policy "公开写" on public.gallery for insert to anon with check (true);
create policy "公开改" on public.gallery for update to anon using (true);
create policy "公开删" on public.gallery for delete to anon using (true);

-- todos 表
CREATE TABLE IF NOT EXISTS public.todos (
  id bigint generated by default as identity primary key,
  text text not null, done boolean default false,
  created_at timestamptz default now()
);
alter table public.todos enable row level security;
create policy "公开读 todos" on public.todos for select to anon using (true);
create policy "公开写 todos" on public.todos for insert to anon with check (true);
create policy "公开改 todos" on public.todos for update to anon using (true);
create policy "公开删 todos" on public.todos for delete to anon using (true);

-- photos + audio 存储桶 RLS (在 Supabase 后台建好 bucket 后跑)
create policy "public upload photos" on storage.objects for insert to anon with check (bucket_id = 'photos');
create policy "public read photos" on storage.objects for select to anon using (bucket_id = 'photos');
create policy "public delete photos" on storage.objects for delete to anon using (bucket_id = 'photos');
create policy "public upload audio" on storage.objects for insert to anon with check (bucket_id = 'audio');
create policy "public read audio" on storage.objects for select to anon using (bucket_id = 'audio');
create policy "public delete audio" on storage.objects for delete to anon using (bucket_id = 'audio');
```

## 7. 用户的硬约束（未来新对话务必遵守）

1. **电脑版本绝对不能动** —— 用户对电脑布局满意。所有响应式只动手机 (@media max-width)。
2. **失败不能退回到 URL 输入** —— Storage/Supabase 失败要自动 base64 兜底, 永远保留"手机选图"体验。
3. **装饰元素（飞机/狐狸/玫瑰/星轨）手机要保留**, 只是缩小, **不隐藏**。
4. **不要瞎删用户没让你删的东西**。
5. **用户是电脑盲人** —— 解释操作要具体到 "点这里"、"输这个命令"。
